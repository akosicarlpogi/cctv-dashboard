# CCTV Network Monitoring System

A Flask and PostgreSQL web application for monitoring camera access, connected devices, user activity, and system security logs through a web-based dashboard.

## Project Description

This project is a web-based CCTV/network monitoring system developed for COMP 012 – Network Administration. It provides a secure login system, a monitoring dashboard, live camera preview, connected device status checking, and real-time system logs.

The system uses Flask for the backend, PostgreSQL for data storage, and HTML, CSS, and JavaScript for the user interface.

## Features

* Secure login system
* Web-based monitoring dashboard
* Live camera preview
* Connected device status monitoring
* Real-time system logs
* Paginated log viewing
* User activity tracking
* Login and logout records
* Failed login attempt records
* Session timeout handling
* IP address logging
* Device information logging
* PostgreSQL database integration
* Cloud deployment support

## Security Features

* Password hashing
* Session management
* Automatic session timeout
* CSRF protection
* Login rate limiting
* Failed login attempt tracking
* Temporary IP blocking
* Temporary account lockout
* Username validation
* Secure database queries
* Security headers
* No-cache protection after logout or timeout
* Discord security alert support
